# Пользователи

Массив пользователей находится в src/users.js

## Примечания

Было создано три редьюсера: сессии, видео, запросов пользователя.

Запросы хранятся в localStorage браузера, туда они сохраняются когда срабатывает экшен-крейтеры.

Для асинхронных запросов используется стандартный redux-thunk
